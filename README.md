# Вытяжная установка В93
# 1. ОБЩИЕ СВЕДЕНИЯ
Объект автоматизации – вытяжная установка, включающая:

Электродвигатель вентилятора (асинхронный, управляемый магнитным пускателем);

Заслонка с двухпозиционным электроприводом на 220В (открытие/закрытие);

Концевые выключатели для контроля крайних положений заслонки;

Тепловое реле для защиты двигателя.
