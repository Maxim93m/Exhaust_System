# Exhaust_System
Вытяжная установка
# 1. ОБЩИЕ СВЕДЕНИЯ
1.1. Объект автоматизации – вытяжная установка, включающая:

Электродвигатель вентилятора (асинхронный, управляемый магнитным пускателем);

Обратный клапан (заслонка) с двухпозиционным электроприводом на 220В (открытие/закрытие);

Концевые выключатели для контроля крайних положений заслонки (опционально);

Тепловое реле для защиты двигателя.

1.2. Назначение системы:

Автоматическое и ручное управление вентилятором и заслонкой;

Обеспечение правильной последовательности включения/выключения для предотвращения обратной тяги и повышения ресурса оборудования;

Контроль фактического состояния вентилятора (по обратному контакту пускателя или датчику тока);

Формирование аварийной сигнализации при нештатных ситуациях;

Отображение состояния оборудования на панели оператора (HMI) или светосигнальной панели.

1.3. Нормативные документы – система должна соответствовать требованиям ПУЭ, ГОСТ 34.601-90 (стадии создания АСУ ТП).
