# ВЫТЯЖНАЯ УСТАНОВКА В93
## Общие сведения
Объект автоматизации – вытяжная установка, включающая:

Электродвигатель вентилятора (асинхронный, управляемый магнитным пускателем);

Заслонка с двухпозиционным электроприводом на 220В (открытие/закрытие);

Концевые выключатели для контроля крайних положений заслонки;

Тепловое реле для защиты двигателя.

##  Перечень входов и выходов контроллера    
|Обозначение|Тип сигнала|Назначение|
|:-|:-|:-|
|**DI_Start**|Дискретный вход|Кнопка «Пуск»|
|**DI_Stop**|Дискретный вход|Кнопка «Стоп»|
|**DI_Open_End**|Дискретный вход|Концевой выключатель заслонки «Открыто»|
|**DI_Closed_End**|Дискретный вход|Концевой выключатель заслонки «Закрыто»|
|**DI_Thermal_Relay**|Дискретный вход|Тепловое реле двигателя|
|**DI_Fan_Running**|Дискретный вход|Обратный контакт магнитного пускателя|
|**DI_Fire**|Дискретный вход|Сигнал "ПОЖАР"|
|**DO_Fan_Contactor**|Дискретный выход|Управление катушкой магнитного пускателя вентилятора|
|**DO_Damper_Open**|Дискретный выход|Управление открытием заслонки (220В на привод открытия)|
|**DO_Damper_Close**|Дискретный выход|Управление закрытием заслонки (220В на привод закрытия)|
|**DO_Alarm_Lamp**|Дискретный выход|Сигнальная лампа «Авария»|
